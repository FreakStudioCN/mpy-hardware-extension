**商业模式锁定（v5）：**

**1 · Skill SaaS / metered**（Cursor-style 订阅或按用付费）
- AI 工具本身就是付费的，跟 Cursor 一样
- 是主营收第一支柱，软件级毛利
- 不是免费引流层

**2 · 模块销售**（每个项目用户买一组）
- 主动模块（ESP32-C3 + 传感/执行）按 SKU 卖
- 硬件级毛利、走量

**3 · 小批量代工**（10-1000 件，可选）
- 创作者商业化时使用
- 服务级毛利

**社区配方层（手机/网页 app）** = 知识倍增器，**不是**营收路径（fork / 学习 / 二次开发是社区免费功能）

**Why（用户 2026-04-29 原话）：** "我说 app store 不是商业模式 那我们主要商业模式不还是 cursor for hardware 吗 帮别人造东西啊"

**用户的核心区分：**
- AppStore distribution ≠ 商业模式（社区层）
- Skill = 商业模式核心（跟 Cursor 一样卖 AI 工具）
- AppStore 不商业不代表 Skill 不商业
