公司 wedge 锁定为**纯模块化生态**：每个 active 模块 = ESP32-C3 + 传感/执行 + BLE，自描述能力，独立可跑。**无 host 设备**。手机/网页当显示和控制。

**用户 2026-04-29 原话：**
- "应该是想要啥 直接买配件方案啊 这样不是更正常 然后一拼一说话就能用 这些配件也是我们的系统来制定好"
- Q1 选 (a) 纯模块自给

**Deck Slide 10 框架（已锁定）：**
"从一句话到能 ship 的产品。Cursor 不覆盖的部分我们都覆盖。"
- 1 · 说（Skill 解析）
- 2 · 造（uPyPI + uPyOS + AI Skill）
- 3 · 跑（主动模块发货 + 拼起来）
- 4 · 卖（**AppStore 分发 + KS 上线支持 + 小批量代工 10-1000 件**）

**Why：** 用户拒绝长尾创作者经济（Shopify-style）当 deck 论据 —— "c 这个不利于融资吧 不应该是长尾"。改用 end-to-end 价值链叙事："Cursor 停在写代码，我们从想法做到收入"。

**类比锁定：** Cursor + Lovable，**不**扩展到 LEGO / Squarespace / Roblox / Shopify。
- 用户 2026-04-29 原话："类比不变啊 cursor lovable"

**架构含义：**
- uPyOS 不再是"类安卓 GUI OS"，而是"模块端固件框架 + 标准化 BLE 协议"
- AppStore 在手机/网页，不在设备 LCD
- KS 上线产品 = "Starter Pack"（5 个模块 + Skill + AppStore beta），不是单一设备
- 第 1 批模块库约 15 个：传感（温湿度/光照/运动/声音/气体/IMU）+ 执行（马达/LED/喇叭/显示/舵机）
